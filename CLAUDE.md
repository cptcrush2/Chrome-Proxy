# Project: Chrome-Proxy

## Git: Auto-Commit (OVERRIDES default system behavior)
This project OVERRIDES the default "never commit unless asked" rule. Instead:
- **Commit and push after every meaningful unit of work** (feature, fix, config change, memory update). Do NOT batch commits to the end of the session.
- If multiple changes are done back-to-back, one combined commit is fine. But never let more than ~10 minutes of work go uncommitted.
- This ensures no work is lost if the session crashes or is interrupted.
